排序算法

发表于 2024-02-19
十种排序算法代码（Python）

冒泡排序 Bubble Sort

每次比较相邻的两个元素

选择排序 Selection Sort

选择最小/最大，放到最前/最后

插入排序 Insertion Sort

从头到尾依次扫描，将元素插入其适当位置

希尔排序 Shell Sort

先将整个序列分割成为若干子序列分别进行插入排序，待整个序列基本有序时，再对整个序列进行插入排序

归并排序 Merge Sort

采用分治法，将大序列划分成很多个小序列（长度为 1 或 2），小序列先自己排序，然后在合并时按照数字大小顺序合并

快速排序 Quick Sort

选择一个基准，将小于基准的数放左边，将大于等于基准的数放右边，然后分别对左右进行快速排序

堆排序 Heap Sort

构建一个大根堆，每次将堆顶的数据放到堆尾，将堆尺寸减 1 并维护堆，重复前面的操作知道完成排序

计数排序 Count Sort

根据最大、最小值开辟数组，统计序列中每个数字出现次数，最后按照数值大小依次输出相应个数的数值

桶排序 Bucket Sort

创建 n 个桶，每个桶有一个数值范围，先将序列中的值放到对应的桶中，然后对每个桶进行排序，最后将桶进行合并

基数排序 Radix Sort

利用了桶的概念，根据键值的每位数字来分配桶
