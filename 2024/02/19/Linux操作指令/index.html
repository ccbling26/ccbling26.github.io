Linux 操作指令

文件管理

chmod

change mode，控制用户对文件的权限的命令

cut

从文件的每一行剪切字节、字符和字段并将这些字节、字符和字段写至标准输出

如果不指定 file 参数，cut 命令将读取标准输入。必须指定 -b、-c 或 -f 标志之一

less

与 more 类似，可以随意浏览文件

more

类似 cat，以一页一页的形式显示，按 SPACE 下一页，按 b 上一页

split

用于将一个文件分割成数个

查看被占用的端口

unclassified

bg

  • 把后台暂停的工作恢复到后台执行

chown

  • change owner，用于设置文件所有者和文件关联组的命令，需要超级用户 root 的权限才能执行该命令

dd

  • 可从标准输入或文件中读取数据，根据指定的格式来转换数据，再输出到文件、设备或标准输出

df

disk free，显示目前在 Linux 系统上的文件系统磁盘使用情况统计

du

disk usage，显示指定的目录或文件所占用的磁盘空间

fg

  • 把后台命令恢复在前台执行

free

显示内存的使用情况，包括实体内存，虚拟的交换文件内存，共享内存区段，以及系统核心使用的缓冲区等

grep

强大的文本搜索工具，用于查找文件里符合条件的字符串或正则表达式

head

用于查看文件的开头部分的内容

  • -q：隐藏文件名
  • -v：显示文件名
  • -c：显示的字节数
  • -n：显示的行数，默认 10 行

ifconfig

显示或设置网络设备

jobs

  • 查看当前终端放入后台的工作
  • -l：列出进程的 PID 号
  • -n：只列出上次发出通知后改变了状态的进程
  • -p：只列出进程的 PID 号
  • -r：只列出运行中的进程
  • -s：只列出已停止的进程

ln

为某一个文件在另外一个位置建立一个同步的链接

kill

用于删除执行中的程序或工作

  • 常用信号
    • 1（HUP）重新加载进程
    • 9（KILL）杀死一个进程
    • 15（TERM）正常停止一个进程（默认信号）

ps

显示当前进程的状态

scp

参考：https://www.runoob.com/linux/linux-comm-scp.html

  • secure copy， Linux 系统下基于 ssh 登陆进行安全的远程文件拷贝命令

  • scp 是加密的，rcp 是不加密的

本地到远程

  • 文件

  • 文件夹

远程到本机

  • 文件

  • 文件夹

su

用于变更为其他使用者的身份，除 root 外，需要键入该使用者的密码

tail

查看文件的末尾部分

tar

参考：https://www.runoob.com/linux/linux-comm-tar.html

  • 用于打包/解压文件

常用选项说明

  • -c：建立新的备份文件
  • -f：指定备份文件名称
  • -x：从备份文件中还原文件
  • -v：显示指令执行过程
  • -z：通过 gzip 指令处理备份文件

打包

解压

telnet

  • 用于远程登录（也可以用于检查网络连接是否正常以及特定端口是否处于打开状态）

top

提供了一个动态的、交互式的实时视图，显示系统的整体性能信息以及正在运行的进程的相关信息

wc

用于计算字数
